Getting Started

Up and running in minutes

Clone the repository, run the bootstrap script, or use the pre-built Docker image.

1

Clone the repository

git clone https://github.com/zegang/ainas.git
2

Run the setup script

bash bootstrap.sh --setup
3

Start the backend

bash bootstrap.sh --backend
4

Launch the frontend

bash bootstrap.sh --frontend

Pre-built images for CPU, NVIDIA GPU, and AMD GPU are available on GitHub Container Registry. The container serves both the backend API and the Flutter web frontend on port 9026.

1

Pull and run (CPU)

docker run -d --name ainas -p 9026:9026 -v ./storage:/app/storage ghcr.io/zegang/ainas-backend:cpu
2

NVIDIA GPU variant

docker run -d --name ainas --gpus all -p 9026:9026 -v ./storage:/app/storage ghcr.io/zegang/ainas-backend:cuda
3

AMD GPU variant

docker run -d --name ainas --device=/dev/kfd --device=/dev/dri --group-add video -p 9026:9026 -v ./storage:/app/storage ghcr.io/zegang/ainas-backend:rocm

Or build locally: bash bootstrap.sh --build-backend-image [cpu|cuda|rocm]
